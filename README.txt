VIBE Photos Folder
==================

Place your images here for the Vibe (photo frame) mode.

File naming:
- Use numbers: 1.jpg, 2.png, 3.gif, etc.
- Maximum 10 images (1-10)
- Supported formats: .jpg, .jpeg, .png, .gif, .webp

Examples:
- 1.jpg
- 2.png
- 3.gif
- 4.webp

The images will be displayed in order and can be swiped through on the phone screen.


Voice command cheatsheet (RU)
=============================

Smart home:
- "включи свет в зале"
- "выключи свет в спальне"
- "выключи всё" / "выключи все устройства" (отправляется команда на все switch-устройства)

Timer:
- "поставь таймер на 2 минуты" (120 секунд)
- "поставь таймер на 15 минут" (900 секунд)

Music:
- "включи музыку"
- "включи радио"
- "покажи музыку"

Modes:
- "покажи погоду"
- "включи ночные часы"
- "включи фоторамку" / "покажи фото" / "вайб"
- "верни обычный режим"
